Moving Save Files between SysMMC and EmuMMC without launching HOS

Preface

Use at your own risk!! It is currently unknown if any of this is bannable in any way, and I also don't know if it will work on future versions of HOS/once your SysMMC and EmuMMC versions no longer match. This process involves working with files on the eMMC directly, and as such backups are strongly recommended. This guide is not for the faint of heart, and you follow it entirely at your own risk. The Guide

Here's what you'll need:

Step 1: Exporting your Saves

  • Put the TegraExplorer.bin file in the bootloader/payloads/ folder.
  • Launch TegraExplorer by injecting hekate, selecting Payloads, and then selecting TegraExplorer.bin.
  • If you wish to export your saves from SysMMC, select Browse EMMC. If you wish to export your saves from EmuMMC, select Browse EMUMMC.
  • Select USER and enter it.
  • Select save and enter it.
  • Select Current folder options (orange option near the top of the list), then Copy to clipboard. Ensure you click Copy and not Move.
  • Back out to the main menu.
  • Select Browse SD.
  • Select Clipboard -> Current folder (orange option near the top of the list) and wait for the operation to finish. This may take some time depending on your system and SD card.

Step 2: Extracting your Saves

  • Unzip SaveExtract.zip to a known location on your computer.
  • Put your prod.keys file in the SaveExtract folder.
  • Move the save folder you got in Step 1 into the SaveExtract folder on your computer. This can either be done by selecting Unmount SD in the TegraExplorer menu and moving the SD card from your switch to your computer, or by selecting Reboot to bootloader/update.bin and moving the folder with hekate's USB Tools.
  • If you are not on Windows, add hactool to your PATH. Windows users do not need to worry about this step.
  • Open a terminal window in the SaveExtract folder and run py extract.py on windows or python3 extract.py on other operating systems.
  • While the script is running, you will likely see a lot of lines starting with [WARN]: Failed to match key "<key_name>", (value "<key_value>"). This is normal and these lines can be safely ignored.
  • Your extracted and decrypted saves will now be in the out folder in the SaveExtract folder. We don't actually need these for this guide, but if you want
  • Ignoring the lines starting with [WARN], sift through the program output and look for lines that have internal_save_number => game_title_id, such as 0000000000000059 => 01008DB008C2C000. It may be beneficial to copy/paste the output into a text editor and use the search or find functions to locate your target Title ID.
  • Make note of the value on the left side of the arrow where the right side matches your desired Title ID. In some cases, there will be multiple. Note down all of them if so. For the purposes of this guide, I will be targeting Pokémon Shield (Title ID: 01008DB008C2C000), and have the following matches in my output: 0000000000000059 => 01008DB008C2C000, 0000000000000166 => 01008DB008C2C000, 00000000000001db => 01008DB008C2C000, and 00000000000001f0 => 01008DB008C2C000.

Step 3: Transplating your Saves

  • (Re-)launch TegraExplorer on your Switch.
  • If you want to move save files from SysMMC to EmuMMC, select Browse EMMC. If you want to move save files from EmuMMC to SysMMC, select Browse EMUMMC.
  • Select USER and enter it.
  • Select save and enter it.
  • One at a time, find the files of interest from Step 2 in this folder, select them, and choose Copy to clipboard. It is very important that you choose Copy and not Move.
  • With this file on your clipboard, go back to the main menu in TegraExplorer and choose Browse SD.
  • Paste the file to your SD card by choosing Clipboard -> Current folder. Repeat the above points for all of the relevant files.
  • One at a time, select the newly-pasted files on the SD card and choose Copy to clipboard.
  • Back out to the main menu.
  • If you are moving save files from Sys to EmuMMC, select Browse EMUMMC. If you are moving save files from EmuMMC to SysMMC, select Browse EMMC.
  • Select USER, then Clipboard -> Current Folder. The file should be pasted below the saveMeta folder.
  • Enter the save folder, then find the file with the same name as the one you just pasted.
  • Select this file and choose Delete file, and confirm by choosing yes.
  • Back out to the previous folder, select the file you pasted there, and choose Move to clipboard. Do not Copy to clipboard this time, use Move instead.
  • Select save again, then Clipboard -> Current folder. The file you just deleted from here will be replaced with he new one.
  • Repeat the above for every file you moved on to the SD card earlier.
  • Once done, you can remove the files from your SD card if you desire.
